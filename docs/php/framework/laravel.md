# Laravel

## 本地化

### 默认时区

在 `config/app.php` 配置文件中编辑：

```php
'timezone' => 'PRC',
```

### 默认区域

在 `config/app.php` 配置文件中编辑：

```php
'locale' => 'zh-CN',
```

### Faker 语言

在 `config/app.php` 配置文件中添加：

```php
'faker_locale' => 'zh_CN',
```

### 多语言包支持
